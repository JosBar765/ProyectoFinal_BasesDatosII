# 🛒 Portal de Compras

Proyecto final del curso **Bases de Datos II**.

Este proyecto consiste en un portal de compras funcional que simula un sistema de e-commerce completo.

---

## 🚀 Características

- 📦 Productos organizados por **categorías**
- ➕ Adición de productos:
  - Masiva (archivos)
  - Interactiva (manual)
- 🖼️ Cada producto cuenta con:
  - Mínimo 1 imagen
  - Máximo 10 imágenes
- 🛍️ Sistema de **carrito de compras**
- 👤 Compras:
  - Como usuario registrado
  - De forma anónima
- 💳 Gestión de perfiles:
  - Almacenamiento de métodos de pago
- 💰 Simulación de pago con **tarjeta de crédito**
- 🎉 Aplicación de descuentos por temporada:
  - Día de la Madre
  - Día del Padre
  - Otras fechas especiales
- 🗄️ Base de datos con más de **2,000 productos**

---

## 🧠 Detalles del sistema

> [!NOTE]
> El sistema permite la asociación de productos a múltiples categorías, tanto en cargas masivas como en inserciones manuales.

> [!IMPORTANT]
> La simulación de pagos no procesa transacciones reales, únicamente representa el flujo lógico de autorización.
